## 핵심 요약

- **수소**는 연소 시 물만 생성되는 무공해 에너지원입니다.
- **그레이수소** → **블루수소** → **그린수소** 순서로 점점 더 깨끗한 생산 방식으로 전환 중입니다.
- 전 세계 수소 경제 규모는 **2050년까지 12조 달러**에 이를 것으로 전망됩니다.
- 세 권의 책은 각각 기술, 투자, 지정학이라는 서로 다른 관점에서 수소의 미래를 조명합니다.

> 자세한 비교 분석은 [도서 비교분석](Prompts/04_도서_비교분석.md)을 참고하세요.
